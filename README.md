# Mathematics 101

A collection of mathematical study materials and course notes in LaTeX format.

## Courses

### 1. Polynomials, Functions and Graphs
Based on the MathTrackX XSeries Program from the University of Adelaide. Covers:
- Numbers (types, arithmetic, inequalities)
- Linear Functions (coordinate plane, slope, equations)
- Polynomials (properties, quadratics, graphs)
- Functions and Graphs (domain, range, continuity, composition)

### 7. Numerical Analysis
Computational methods for solving mathematical problems. Covers:
- Errors and Approximations
- Root Finding Methods (Bisection, Newton-Raphson, Secant, Fixed-Point)
- Interpolation (Lagrange, Newton's Divided Differences)
- Numerical Differentiation and Integration
- Systems of Linear Equations
- Numerical Solutions of ODEs

### Graph Theory Report
A comprehensive report on graph theory concepts and applications.

## Structure

Each course folder contains:
- `study-material.tex` - Main study material
- `exercises.tex` - Practice problems (where applicable)
- `cheatsheet.tex` - Quick reference (where applicable)

## Building

Compile any `.tex` file using:
```bash
pdflatex filename.tex
```
